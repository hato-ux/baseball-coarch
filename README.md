# ⚾ BASEBALL COACH PRO

野球選手・コーチのための科学的指導マニュアルアプリ。

## 📱 デプロイ方法（Vercel）

### 方法①：Vercel CLIを使う
```bash
npm install -g vercel
cd baseball-coach-app
vercel
```

### 方法②：GitHubからデプロイ
1. このフォルダをGitHubにpush
2. [vercel.com](https://vercel.com) でGitHubリポジトリを選択
3. デプロイボタンを押すだけ

### 方法③：Vercel Web UIにドラッグ&ドロップ
1. [vercel.com/new](https://vercel.com/new) にアクセス
2. このフォルダをそのままドラッグ&ドロップ
